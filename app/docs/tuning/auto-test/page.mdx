# Auto Test
---

<Steps>

### Clear Out an Open Space
Before running this OpMode, make sure you have an environment with enough space to run the Auto Test to ensure safety and prevent damage.

### Run the OpMode
Select the **Auto Test** OpMode on your Driver Station.

### The Autonomous Path
* Your robot will start at `Pose(0, 0, 0)` - AKA the centre of the field in Apex Coordinates -
 and will perform a full 360 degree tornado spin on its way to `Pose(45, 25, 25)` by interpolating heading using a custom heading function.
* The path's customizable `exampleCallback()` function will trigger exactly halfway through this path. You may fill in a custom action in this
method's body to test it.
* Finally, the robot will perform a test turn holding the exact Pose of the end position of the previous path.
* If all of this runs as intended, that means that Apex Pathing is now 100% ready to go and you are ready to Path your way to the Peaks!

</Steps>
