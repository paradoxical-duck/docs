## Tuning your Follower
* In order to tune your follower, a single, unified tuner called the [Follower Tuner](/docs/tuning/follower-tuner)
* This tuner is capable of tuning all of the necessary follower constants *and* saving the values to the `FollowerConstants` file
* It also has the flexibility of a manual mode, enabling users to tune the Constants *manually* using Panels
* This is the only tuner which needs to be run for the follower, and doesn't even require you to input any values into the Constants file!
* Make sure to thoroughly read the instructions before running this tuner and to follow the instructions on the telemetry of your Driver Hub
