# Drivetrain Configs
___



<Callout type="warning">
It's imporant to change the drivetrain listed in constants you can see it is `BaseDrivetrainConfig<DRIVETRAIN TYPE.Config>`
This is important because if it's the wrong drivetrain, the constants file will not compile
</Callout>
## Mecanum Drive
The mecanum drivetrain takes multiple arguments in the constants file: 
* Motor names
* Motor directions
* Maximum power
* Robot centric drive
```java
@Override
    public BaseDrivetrainConfig<Mecanum.Config> setDrivetrainConstants() { // Any baseDrivetrainConfig child
        return new Mecanum.Config()
                //front left motor name + direction 
                .setFrontLeftMotor(new MotorFactory("frontLeftMotor"))
                //back left motor name + direction
                .setBackLeftMotor(new MotorFactory("backLeftMotor"))
                //front right motor name + direction
                .setFrontRightMotor(new MotorFactory("frontRightMotor").reverse())
                //back right motor name + direction
                .setBackRightMotor(new MotorFactory("backRightMotor").reverse())
                //robot centric or field centric
                .setRobotCentric(true)
                //max power threshold
                .setMaxPower(1.0);
    }
```


## Swerve Drive
The swerve drivetrain takes multiple arguments in the constants file:
* Drive motor names
* Drive motor directions
* Turn servo names
* Analog encoder names
* Trackwidth and wheelbase measurements
* maximum power
* angular offsets for each module
Note: The swerve drivetrain constants require declarations grouped together in modules.
```java
@Override
    public BaseDrivetrainConfig<CoaxialSwerve.Config> setDrivetrainConstants() { // Any baseDrivetrainConfig child
        return new CoaxialSwerve.Config()
                //front left motor name 
                .setFrontLeftMotor(new MotorFactory("frontLeftMotor"))
                //back left motor name
                .setBackLeftMotor(new MotorFactory("backLeftMotor"))
                //front right motor name
                .setFrontRightMotor(new MotorFactory("frontRightMotor"))
                //back right motor name
                .setBackRightMotor(new MotorFactory("backRightMotor"))
                //servo names
                .setServoNames("frontLeftServo", "frontRightServo", "backLeftServo", "backRightServo")
                //encoder names 
                .setEncoderNames("flEncoder", "frEncoder", "blEncoder", "brEncoder")
                //robot trackwidth
                .setTrackWidth(Dist.fromIn(14))
                //robot wheelbase
                .setWheelbase(Dist.fromIn(14))
                //max power threshold
                .setMaxPower(1)
                //robot centric or field centric
                .setRobotCentric(true);
    }
```
