## Why?
To build an advanced, accessible pathfollowing solution for FTC that eliminates common barriers to entry by 
delivering easy to use, top-tier autonomous performance alongside clear documentation.

## What are the Benefits?
* Easy to understand documentation that has both basic and advanced concepts
* An intuitive path generation API that gets its power from simplicity and flexibility with other libraries and tools
* B-Spline based path following for smooth, performant, and customizable trajectories
* Major loop time optimizations during Autonomous and TeleOp
* OnBot Java environment support
* Mostly automatic tuning
* Powerful visualizer with simulation, global poses, and code generation for easy debugging and development
* Clean and self documenting code with a focus on readability and education
* Plans for updates in the future to add features such as a command based framework, more advanced path following algorithms, and more!

## When is This Being Released?
Apex Pathing is currently in the development and is not officially released. We're aiming to publish the first version 
of Apex Pathing in late July, giving teams enough time to experiment and prepare before the 2026-2027 FTC BIOBUZZ season begins. Join our 
[Discord server](https://discord.gg/zQyJ5NCtD) to receive updates about Apex releases and chat with our developers and testers!
